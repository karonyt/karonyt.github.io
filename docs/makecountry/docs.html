導入ガイド

このドキュメントでは、プロジェクトの使い方について詳しく説明します。

インストール方法

以下のサイトから最新のMakeCountryをダウンロードできます

https://github.com/karonyt/MakeCountry/releases/latest

mcpackファイルをダウンロードしてください

ダウンロードした後、ダウンロードしたmcpackファイルを開いてください(Minecraftで開く)
その後、ワールド設定でベータ APIをオンにしてからワールドに導入してください
導入後は以下のコマンドをチャットで実行してください 

/mc:setup

これにてインストール完了です
